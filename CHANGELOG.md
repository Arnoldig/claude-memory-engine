# История изменений

[English](CHANGELOG.en.md)

Здесь перечислены заметные изменения проекта. Формат по образцу [Keep a Changelog](https://keepachangelog.com/), нумерация версий по [семантическому версионированию](https://semver.org/).

## [0.6.0] — 2026-06-21
### Добавлено
- `claude-memory uninstall` — убирает движок из проекта (регистрации хуков, обёртку, конфиг и вшитую копию при установке через git); ваши уроки не затрагиваются.
- Английская версия `README.en.md` рядом с основным `README.md`.
### Изменено
- README переписан для широкой аудитории (от общего к частному; текст отделён от команд).
- Вывод `install.sh` и команд переведён на английский.
### Исправлено
- `install`/`uninstall` больше не падают на «валидном, но не-объектном» `settings.json` или конфиге.
- Версия пакета синхронизирована (`pyproject.toml` отставал от кода).

## Ранние выпуски (2026-06-17 – 2026-06-18)
- **0.5.2** — срок хранения архивных уроков + `archive_prune`.
- **0.5.1** — диагностика: сигнал об уроках с пустым именем.
- **0.5.0** — pip-пакет + установка одной командой `claude-memory init` (плюс `doctor`, `config`, `--version`).
- **0.4.0** — SQLite-кэш для поиска (тот же порядок выдачи, быстрее на больших наборах).
- **0.3.0** — самопроверка конфига и «пульс здоровья» (битые перекрёстные ссылки, подсказка о дублях).
- **0.2.0** — мультиязычные сообщения (английские по умолчанию + переопределение в проекте), полностью настраиваемое поведение, поддержка worktree.
- **0.1.0** — первый движок: уроки с frontmatter, авто-оглавление, офлайн-поиск, уроки по пути файла, авто-обслуживание, страховки для параллельных сессий и суб-агентов.
